# Mongodb Cron Backup

Run schedule to create full backup for mongodb database, save it on server and send copy to mail

# Steps to install
 1. Rename **.env.example** to **.env** and update values
 2. Install packages via terminal **npm install**

# Run via CLI
```
node index.js
```
